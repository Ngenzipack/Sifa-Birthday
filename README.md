# Sifa Birthday (GitHub Pages)

A one-page birthday site for Sifa with background music and a printable QR code.

## Live URL

- https://ngenzipack.github.io/Sifa-Birthday/

## Local files

- `index.html`: birthday page
- `styles.css`: visual theme and motion
- `script.js`: tap-to-play audio logic
- `assets/happy-birthday.mp3`: background song asset
- `qr/`: print-ready QR artifacts

## Song source

- `assets/happy-birthday.mp3` copied from local file provided by the sender.
